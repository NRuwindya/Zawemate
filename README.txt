Project Title: Zawemate – Financial Management & Budgeting Informational Website
Developer: Ruwindya Niunhellage

Project Overview:
Zawemate is an informational website focused on financial management and budgeting, created to support the UN Sustainable Development Goal: Quality Education. The website provides lessons, comparison tables of budgeting methods, and guidance to help users improve their financial literacy.

Pages Included:
- home.html       -> Hero section, mission statement, and navigation
- info.html       -> Detailed lessons and content sections
- table.html      -> Comparison table with budgeting strategies
- contact.html    -> Contact form with radio buttons, checkboxes, and validation
- about.html      -> Founder profile with image, skills, and contact info

Technologies Used:
- HTML5 for semantic page structure
- CSS3 (Flexbox & Grid) for layout, responsiveness, and animations
- Fully responsive design for desktop, tablet, and mobile devices

Project Highlights:
- Clear and accessible financial education content
- Pixel-perfect implementation of Figma designs
- Responsive layout with left-aligned images and right-aligned text
- Accessible fonts, readable text sizes, and sufficient color contrast
- Smooth CSS transitions and hover effects
- Semantic HTML with alt attributes for all images
